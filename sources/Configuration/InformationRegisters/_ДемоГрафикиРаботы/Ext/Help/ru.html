Регистр сведений предназначен для примера использования ряда подсистем.

Для подсистемы Обмен данным является примером регистра сведений, который синхронизируется с другими информационными базами.

Для подсистемы Поиск и удаление дублей регистр сведений выступает в качестве источника временной схемы исходных данных, участвующих в регистре расчета Демо: Основные начисления сотрудников.
